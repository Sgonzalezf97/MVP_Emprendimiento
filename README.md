### Instalacióm

Primero, se clona el siguiente repositorio:

```bash
git clone https://github.com/Sgonzalezf97/MVP_Emprendimiento.git
```

Segundo, descargar los paquetes: 

```
flutter pub get
```
Tercero, correr la aplicación:

```
flutter run
```
